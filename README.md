# 项目简介

本项目是基于深度学习YOLOv8算法的工地安全帽佩戴检测系统。主要功能模块包括实时视频流处理、目标检测、安全帽佩戴识别等。技术特点方面，系统采用先进的YOLOv8算法，在保证检测速度的同时，提高了准确率。该系统可应用于工地的安全监管，对未佩戴安全帽的行为进行实时识别与预警，以减少安全事故发生，保障工人生命安全。


## 核心功能
- 实时视频流处理
- 目标检测
- 安全帽佩戴识别
- 未佩戴安全帽行为实时识别与预警

## 适用场景
- 工地安全监管
- 安全事故预防
- 工人生命安全保障

## 技术特点
- 基于YOLOv8深度学习算法
- 检测速度快
- 检测准确率高
